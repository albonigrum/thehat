Hello, it's ShlyapaHelper help :)
ShlyapaHelper created for one reason - help with fair "each with each" variant of shlyapa game
But ShlyapaHelper can helps with original and not fair "each with each" variants of game.

To configure current game choose "Set current config". !!! this configures don't save after game !!!

To show current configures choose "Show current configures"

To load configures from file choose "Load config from file"
                              !!! Example of config file you can find by path "config/game.yaml" !!!

To Exit - "Exit"

To start game - "Start"

IMPORTANT: in game you can enter some keys. Key 'h' - tell you more about keys in game.
